# MoreClassChildren

Ancien projet expérimental PHP visant à explorer la composition dynamique d'objets et la délégation de méthodes via `__call`.

L'objectif était de tester une alternative à l'héritage multiple en PHP, en agrégeant plusieurs instances dans une classe façade.

Ce projet est conservé comme expérimentation personnelle et n'est pas destiné à une utilisation en production.
